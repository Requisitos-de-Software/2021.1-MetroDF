    Rich Pictures

    Introdução

    Rich picture é uma ferramenta para analisar problemas e expressar ideias. Basicamente, é o desenho de uma situação que ilustra os principais elementos e relações que precisam ser considerados ao tentar intervir a fim de criar alguma melhoria. É composto por imagens, textos, símbolos e ícones, que servem para ilustrar graficamente a situação.
