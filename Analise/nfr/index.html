    Verificação - NFR Framework

    Contexto

    • Documento avaliado: NFR Framework
    • Técnica utilizada: Inspeção com Leitura Check-List
    • Inspetor: Hérya Rodrigues

    Checklist

    1. Todas as informações necessárias estão presentes?
    2. As informações são consistentes e concisas?
    3. Há apenas informações necessárias?
    4. O fluxo do diagrama está bem representado?
    5. A linguagem utilizada é compreensível ao público?
    6. As setas indicam o fluxo corretamente?
    7. Existe legenda?
    8. Os elementos utilizados estão na legenda?
    9. Os softgoals são condizentes com o contexto?
    10. A propagação é coerente?
    11. Os relacionamentos são claros e coerentes?
    12. Possui contribuições apenas onde é necessário?
    13. A ortografia está correta?
    14. As nuvens foram utilizadas corretamente?
