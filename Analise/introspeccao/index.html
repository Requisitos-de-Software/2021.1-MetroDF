    Verificação - Introspecção

    Contexto

    • Documento avaliado: Introspecção
    • Técnica utilizada: Inspeção com Leitura Check-List
    • Inspetor: Felipe Correia

    Checklist

    1. O requisito está transcrito de forma clara?
    2. Todos as informações necessarias estão presnetes?
    3. Todos os termos importantes possuem lexicos?
    4. Todos os léxicos estão linkados?
